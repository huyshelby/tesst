# Product Images

Thư mục chứa hình ảnh sản phẩm cho admin dashboard.

## Cấu trúc

```
pictures/
├── iphone/          # Hình ảnh iPhone
├── laptop/          # Hình ảnh Laptop
├── tablet/          # Hình ảnh iPad/Tablet
├── watch/           # Hình ảnh Apple Watch
├── audio/           # Hình ảnh AirPods, tai nghe
└── accessories/     # Hình ảnh phụ kiện
```

## Copy hình ảnh từ phone-app

Để sử dụng hình ảnh từ phone-app, chạy lệnh sau:

### Windows (PowerShell)
```powershell
# Copy tất cả hình ảnh
Copy-Item -Path ..\phone-app\public\pictures\* -Destination .\public\pictures\ -Recurse -Force
```

### Linux/Mac
```bash
# Copy tất cả hình ảnh
cp -r ../phone-app/public/pictures/* ./public/pictures/
```

## Upload hình ảnh mới

1. Đặt hình ảnh vào thư mục phù hợp (iphone/, laptop/, etc.)
2. Đặt tên file theo format: `product-slug.png` hoặc `product-slug.jpg`
3. Kích thước đề xuất: 800x800px hoặc 1200x1200px
4. Format: PNG, JPG, WebP
5. Tối đa: 5MB/file
