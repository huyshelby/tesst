# Uploaded Product Images

Thư mục chứa hình ảnh sản phẩm được upload từ admin dashboard.

## Thông tin

- **Nguồn**: Upload từ admin dashboard
- **URL**: `http://localhost:4000/pictures/uploaded/filename.png`
- **Giới hạn**: 5MB/file
- **Format**: JPG, PNG, WebP

## Cấu trúc tên file

Format: `{normalized-name}-{timestamp}-{random}.{ext}`

Ví dụ: `iphone-15-pro-1766123456789-123456789.png`
